# Weave brand assets

SVG logos used by the Weave (Knowledge Fabric) platform UI.

| File | Use |
|------|-----|
| `weave-logo.svg` | Icon mark only (matches `frontend/src/components/WeaveLogo.tsx`) |
| `weave-logo-app.svg` | Square app icon with dark rounded background |
| `weave-logo-wordmark.svg` | Icon + “Weave” text for decks and headers |
| `weave-flyer.html` | Interactive flyer (open in browser) |
| `weave-flyer.png` | Flyer image (900×3200) |
| `weave-flyer.jpg` | Flyer JPEG export |
| `weave-flyer-hd.png` | High-DPI flyer (1800×6400, print quality) |
| `weave-flyer-hd.jpg` | High-DPI JPEG |

Gradient: violet `#8b5cf6` → fuchsia `#d946ef` → cyan `#06b6d4`.

### Flyer
Open from repo root:
```bash
open Weave/weave-flyer.html
```
Uses `weave-logo.svg` + `../frontend/public/TCS-logo-white.svg`. Best exported via **Print → Save as PDF** (enable background graphics).
